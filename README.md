# Airline_Project(DSA Major Project)

-> The project includes a UI, also known as GUI (Graphical User Interface), which consists of buttons, text fields, and other graphical elements used to interact with the user.

-> The Technology used for making Airline Reservation System is Java Swing library,which provides various features and functionalities.

-> In this project includes both Front end and back end. Java Swing includes components such as JFrame, JButton, and JPanel, which will be used to build the project's front end.

-> GUI needs to be connected with the database, for that JDBC driver used to connect Java with MySQL for the project.
